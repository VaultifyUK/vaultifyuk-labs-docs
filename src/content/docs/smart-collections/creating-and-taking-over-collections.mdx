Smart Collections can manage a collection in two ways: create a new one from scratch, or take over an existing collection already in your store. Both result in a collection whose membership and sort order are controlled automatically.

## Setting up a collection

<Tabs>
  <TabItem label="Create new">

When you create a collection through Smart Collections, it creates a matching Shopify collection and immediately takes ownership of its membership and sorting.

<Steps>

1. Click **New Collection** in the Smart Collections dashboard
2. Select **Create new collection**
3. Enter a title — this becomes the Shopify collection title. Optionally add a description, SEO title, SEO description, and handle.
4. Configure your membership rules (see [Rule-Based Collections](/smart-collections/rule-based-collections/))
5. Click **Save and sync**

</Steps>

The collection appears in your Shopify store immediately. Smart Collections runs an initial sync to populate it based on your rules.

:::note
After creation, Shopify owns the collection's title, description, image, SEO fields, and handle. You can edit any of these from your Shopify admin or from the collection's detail page in Smart Collections. Smart Collections only controls which products are in it and their order.
:::

  </TabItem>
  <TabItem label="Take over existing">

You can hand off an existing manual Shopify collection to Smart Collections without disrupting it. The collection stays live in your store throughout the process.

:::note
Only manual Shopify collections can be taken over. Shopify automated collections (those with Shopify-managed rules) are not supported — Smart Collections will reject them because Shopify's rule engine conflicts with its own membership management. To migrate a Shopify automated collection, create a new Smart Collections collection with equivalent rules instead.
:::

<Steps>

1. Click **New Collection** in the Smart Collections dashboard
2. Select **Take over existing collection**
3. Search for the collection by name and select it
4. Configure your membership rules
5. Click **Save and sync**

</Steps>

Smart Collections sets the collection to manual sort order (required for automated management) then runs a baseline import before the first evaluation.

:::note
When you take over a collection, Smart Collections first records its current product set. This import runs in the background and must complete before the first sync can run — you will see a progress indicator on the collection detail page while it is in progress. For large collections this may take a minute or two.
:::

:::caution
When the first sync runs after takeover, Smart Collections will add or remove products based on your rules. Products that do not match your rules will be removed on the next sync. Review your rules carefully before saving.
:::

### What changes when you take over a collection

| Area | Before takeover | After takeover |
| --- | --- | --- |
| Membership | Managed manually | Managed by Smart Collections rules |
| Sort order | Shopify-controlled | Smart Collections-controlled (set to Manual in Shopify) |
| Title, description, image, SEO | Shopify-owned | Shopify-owned (unchanged) |
| Automatic updates | None | Automatic sync on product changes |

  </TabItem>
</Tabs>

## The collection detail page

Once a collection is managed, its detail page shows:

- Current sync status and last sync time
- Rule configuration
- Sort configuration and pin settings
- Sync history (last 10 syncs, with products added, removed, and reordered)

A **Sync now** button lets you trigger an immediate re-evaluation and re-sort at any time.

## Stopping management

### Release a collection

Releasing a collection stops Smart Collections from managing it without deleting it from Shopify. The collection remains in your store with its current products and positions intact.

:::note
Releasing restores the sort order that was active when you took over. Only the sort mode changes — products are not moved or deleted.
:::

<Steps>

1. Open the collection in Smart Collections
2. Scroll to the **Danger Zone** section
3. Click **Stop managing collection**
4. Confirm in the dialog

</Steps>

After release, the collection returns to full merchant control. You can take it over again at any time.

:::caution
Only the store owner can release or delete a collection. Staff members with Shopify admin access can manage rules, sort settings, and syncs, but cannot perform these destructive actions.
:::

### Delete a collection

Deleting a collection permanently removes it from Shopify.

<Steps>

1. Open the collection in Smart Collections
2. Scroll to the **Danger Zone** section
3. Click **Delete collection**
4. Confirm in the dialog

</Steps>

:::caution
This permanently removes the Shopify collection and cannot be undone. Sync history is preserved in Smart Collections for reference, but the collection and all its storefront associations are gone.

If you only want to stop automating a collection without removing it, use **Stop managing collection** instead.
:::

## If a collection is deleted in Shopify admin

If a managed collection is deleted from Shopify admin rather than through Smart Collections, the app typically detects this via a webhook and removes it from your dashboard. Sync history is preserved.

If the collection still shows an error status after being deleted in Shopify, release it from the Danger Zone — Smart Collections detects the collection no longer exists and cleans up without error.
